# Moonlit Match Fast Sales System

## Current Priority

The current priority is fastest compliant affiliate sales possible through Pinterest traffic.

Moonlit Match is being built as a long-term astrology and soulmate guidance brand, but the first monetization goal is affiliate conversion through high-intent Pinterest traffic.

## Core Funnel

Pinterest Pin -> Moonlit Match Bridge Page -> ClickBank Affiliate Offer

## Hard Rule

Pinterest pins must not link directly to affiliate offers.

All Pinterest traffic must go to a Moonlit Match bridge page first.

## Pinterest Disclosure Rule

Any Pinterest pin linking to a Moonlit Match affiliate bridge page should include a clear affiliate disclosure.

For Pinterest pin descriptions that promote affiliate bridge pages, do not use hashtags. Place this disclosure sentence at the end of the description:

```text
Moonlit Match may receive a small commission if you choose to continue through this link, at no extra cost to you.
```
